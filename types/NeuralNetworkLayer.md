## NeuralNetworkLayer

This type represents a neural network layer instance.

A layer is a callable object that takes as input one or more [NeuralNetworkTensors](types/NeuralNetworkTensor) and that outputs one or more [NeuralNetworkTensors](types/NeuralNetworkTensor).
