## Predict

[foto card]

Generates output predictions for the input samples, for a given model.

### Inputs

* **Model** --- [DynamicObject](types/DynamicObject.html)  
  The model.

* **X** --- [Tensor](types/Tensor.html)  
  The input samples.

### Outputs

* **Y'*** --- [Tensor](types/Tensor.html)  
  The output predictions.
